What TokenLens detects
Zero-cache projects
Projects with cache_write_tokens = 0. Every token of context is paid at full input price, every single turn. One config change fixes it.
Low cache hit rate
Projects writing to cache but not benefiting — cache expires before reuse. Recommends longer-lived cache with ephemeral_1h vs ephemeral_5m.
Savings estimate
Every recommendation includes a dollar savings estimate. Not "consider caching" — "implement system prompt caching here to save ~$127/month."

Common root causes
Large file dump without chunking
Engineer pastes entire codebase into every message. 50K tokens/turn × 30 turns = $164 session. Fix: implement context management.
Runaway tool-call loop
Agent repeatedly calls the same tool without convergence. Each iteration adds context. Fix: add explicit stop conditions.
No context pruning
Session grows to 100+ messages with full history. Context window fills, costs escalate. Fix: summarise and compress every N turns.
